create (...)
Signature: getNow:Job<int64> -> defaultMaxAge:int64 -> maxAgeOnTimeout:int64 -> MaxAgeOnFailure:int64 -> CacheExpirationParameters

Creates a CacheExpirationParameters value with a custom implementation for obtaining the current UNIX timestamp. Also specifies the default max age in seconds to use for any token which doesn't specify its own expiration as well as the how long to cache timeouts and API failures.
